자바 기초 가이드

1. 자바란?

자바는 객체 지향 프로그래밍 언어로, 플랫폼 독립적이며 "한 번 작성하면, 어디서나 실행된다"는 특징을 가지고 있습니다.

2. 자바 개발 환경 설정

  • JDK (Java Development Kit) 설치
  • IDE 설치 (Eclipse, IntelliJ IDEA 등)
  • 환경 변수 설정

3. 기본 문법

첫 번째 자바 프로그램

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

4. 변수와 데이터 타입

  • 기본 데이터 타입
    • 정수형: byte, short, int, long
    • 실수형: float, double
    • 문자형: char
    • 논리형: boolean
  • 참조 데이터 타입
    • 클래스 (String 등)
    • 배열
    • 인터페이스

5. 제어문

  • 조건문: if, switch
  • 반복문: for, while, do-while

6. 객체 지향 프로그래밍

  • 클래스와 객체
  • 상속
  • 다형성
  • 캡슐화
  • 추상화

7. 예외 처리

try {
    // 예외가 발생할 수 있는 코드
} catch (Exception e) {
    // 예외 처리 코드
} finally {
    // 항상 실행되는 코드
}

8. 컬렉션 프레임워크

  • List: ArrayList, LinkedList
  • Set: HashSet, TreeSet
  • Map: HashMap, TreeMap
